# PetShop UI

Angular frontend for the PetShop platform.

## Description
Implements the user interface for browsing products, managing carts, and placing orders.  
Uses Auth0 for authentication and communicates through the API gateway.

## Tech Stack
- Angular  
- Angular Material  
- Auth0 SPA SDK
